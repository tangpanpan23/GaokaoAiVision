# 灯塔志愿微信小程序

基于高考志愿填报系统的微信小程序前端实现，提供完整的志愿填报辅助功能。

## 📱 项目简介

"灯塔志愿"是一款基于人工智能的高考志愿填报辅助小程序，致力于为考生提供公平、透明、个性化的志愿填报指导。

### 🎯 核心特色

- **信息公平**: 所有核心功能永久免费
- **AI智能**: 集成大语言模型提供志愿推荐
- **数据透明**: 基于官方录取数据
- **真实分享**: 学长学姐的亲身经历

## 🚀 快速开始

### 环境要求

- 微信开发者工具 1.06.2307260 或以上版本
- Node.js 16.0.0 或以上版本（可选，用于构建优化）

### 运行步骤

1. **克隆项目**
```bash
git clone <repository-url>
cd GaokaoAiVision
```

2. **打开微信开发者工具**

3. **导入项目**
   - 点击"导入项目"
   - 选择项目路径：`GaokaoAiVision/miniprogram`
   - AppID：填写你的小程序AppID（或选择"测试号"）
   - 点击"确定"

4. **编译运行**
   - 点击工具栏的"编译"按钮
   - 或按 Ctrl + S 保存自动编译
   - 在模拟器中查看效果

## 🎨 功能模块

### ✅ 已实现功能

#### 1. 首页
- **功能入口**: 功能导航（部分功能开发中状态）
- **数据统计**: 平台用户和数据统计（模拟数据）
- **公告通知**: 最新志愿填报资讯
- **用户登录**: 微信授权登录（模拟）

#### 2. 志愿推荐 ⭐
- **智能分析**: 基于分数、位次、科目、兴趣的AI推荐
- **三维规划**: 冲刺、稳妥、保底三层志愿方案
- **详细解读**: 每个学校的优势和注意事项
- **匹配度评分**: 专业与个人兴趣的匹配分析
- **表单验证**: 完整的输入验证和错误提示

### 🚧 开发中功能

#### 3. 分数查询
- **历年数据**: 查询各学校各专业历年录取分数线
- **位次换算**: 分数与位次的对应关系
- **趋势分析**: 专业录取分数趋势图表

#### 4. AI咨询
- **智能问答**: 志愿填报相关问题咨询
- **专业解答**: 基于大数据的专业建议
- **会话记录**: 咨询历史记录保存

#### 5. 学长分享
- **真实经历**: 过来人的校园生活分享
- **专业点评**: 对各专业的深度评价
- **经验交流**: 学习方法和就业建议

#### 6. 职业测评
- **性格测试**: MBTI职业性格分析
- **兴趣测评**: 霍兰德职业兴趣测试
- **职业建议**: 基于测评结果的职业规划建议

#### 7. 个人中心
- **档案管理**: 高考信息和个人资料
- **收藏记录**: 收藏的学校和专业信息
- **使用历史**: 志愿推荐和查询历史

## 📊 数据流程

### 志愿推荐流程
1. **信息录入**: 用户填写高考分数、位次、选科、兴趣
2. **数据验证**: 前端表单验证和数据校验
3. **AI分析**: 调用后端AI服务进行智能分析
4. **结果展示**: 三维志愿方案的可视化展示
5. **交互反馈**: 用户可以调整参数重新生成

### 用户认证流程
1. **微信授权**: 调用wx.login获取code
2. **服务端验证**: 后端验证微信code并返回token
3. **状态管理**: 前端保存登录状态
4. **自动登录**: 应用重启时自动恢复登录状态

## 🔍 开发调试

### 开发者工具调试
1. **控制台**: 查看日志输出和错误信息
2. **Network**: 监控网络请求和响应
3. **Storage**: 检查本地存储数据
4. **WXML**: 实时查看页面结构变化

### 真机调试
1. **预览码**: 生成二维码在手机上预览
2. **远程调试**: 通过USB连接进行真机调试
3. **性能监控**: 监控内存使用和渲染性能

### 常见问题
- **样式不生效**: 检查WXSS文件是否正确引入
- **数据不更新**: 确认setData调用和数据绑定
- **API调用失败**: 检查网络配置和接口地址
- **页面跳转异常**: 验证路由配置和页面路径

## 📋 开发规范

### 命名规范
- **文件命名**: 小写字母 + 下划线 (user_info.js)
- **变量命名**: 驼峰命名 (userInfo)
- **常量命名**: 全大写 + 下划线 (API_BASE_URL)

### 代码规范
- **缩进**: 2个空格
- **分号**: 必须使用分号
- **注释**: 重要函数和复杂逻辑必须注释
- **错误处理**: 所有异步操作都要处理错误
