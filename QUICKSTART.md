# 🚀 灯塔志愿小程序 - 快速开始指南

## 问题解决方案

### ❌ 错误："未找到 pages/score-query/index.wxml 文件"

**问题原因**: app.json 中定义了未实现的页面路径

**解决方案**: ✅ **已修复**
- 移除了未实现的页面路径
- 只保留已完成的页面：首页和志愿推荐页
- 未实现的功能在首页显示"功能开发中"提示

## 📱 立即运行步骤

### 1. 打开微信开发者工具
```bash
# 启动脚本检查项目状态
./start-miniprogram.sh
```

### 2. 导入项目
- **项目路径**: `GaokaoAiVision/miniprogram`
- **AppID**: 使用"测试号"或你的小程序AppID
- **项目名称**: 灯塔志愿

### 3. 编译运行
- 点击工具栏的"编译"按钮
- 或按 `Ctrl + S` 自动编译
- 在模拟器中查看效果

## 🎯 体验核心功能

### ✅ 已实现功能

#### 🏠 首页
- 查看功能导航和数据统计
- 点击"志愿推荐"进入核心功能
- 其他功能显示"开发中"提示

#### 🎯 志愿推荐 ⭐
1. **填写信息**:
   - 选择高考省份
   - 选择分数类型（文科/理科/综合改革）
   - 输入高考分数和位次
   - 选择选考科目
   - 选择兴趣标签（最多3个）

2. **AI分析**:
   - 点击"生成推荐"
   - 系统模拟AI分析过程（2秒）
   - 显示三维志愿方案

3. **查看结果**:
   - **冲**: 高分冲刺理想学校
   - **稳**: 稳妥选择有把握的学校
   - **保**: 保底确保录取的学校

4. **详细信息**:
   - 每个学校显示录取分数线
   - 专业信息和优势分析
   - 匹配度评分和建议

## 🔧 技术架构

### 前端技术栈
- **框架**: 原生微信小程序
- **语言**: JavaScript ES6+
- **样式**: WXSS + CSS变量
- **数据**: 模拟API（本地数据）

### 核心文件结构
```
miniprogram/
├── app.json          # 应用配置 ✅
├── app.js           # 全局逻辑 ✅
├── app.wxss         # 全局样式 ✅
├── pages/
│   ├── home/        # 首页 ✅
│   └── volunteer-suggestion/  # 志愿推荐 ✅
├── utils/
│   ├── api.js       # API接口 ✅
│   └── util.js      # 工具函数 ✅
└── styles/          # 样式系统 ✅
```
